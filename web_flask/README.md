# AirBnB Clone - Web Framework

This project implements a web framework using Flask for the AirBnB clone.

## Project Tasks:
1. Hello Flask!
2. HBNB
3. C is fun!
4. Python is cool!
5. Is it a number?
6. Number template
7. Odd or even?
8. List of states
9. Cities by states
10. States and State
11. HBNB filters
12. HBNB is alive!

## Requirements:
- Python 3.4.3
- Flask
- PEP 8 style (version 1.7)
- All files must be executable
- W3C compliant HTML/CSS files
